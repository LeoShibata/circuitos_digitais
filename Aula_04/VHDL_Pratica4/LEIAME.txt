COMO USAR
---------
Logisim-evolution:
  Coloque todos os circuitos (porta-a-porta NÃO otimizado, otimizado e bloco
  VHDL) dentro de UM ÚNICO projeto Logisim-evolution. Para cada bloco VHDL,
  use "Add VHDL Entity" e cole o conteúdo do arquivo correspondente.

Quartus (EPM240):
  Exporte o VHDL de cada bloco do Logisim ou use estes arquivos .vhd direto.
  Para o exercício 4c, crie um projeto com: divider.vhd, debounce.vhd,
  ex4b_ff_jk_logica.vhd (ou ex4c_ff_t_logica.vhd) e o top-level correspondente.
  Ajuste o pinout:
    - clk50   -> pino 12 (50 MHz do kit)
    - key_clk -> botão CLK da placa adicional
    - J,K ou T, CLRN, PRN -> botões da placa adicional
    - Q, QN   -> pontos decimais dos displays 7s

TABELAS VERDADE (para o relatório)
-----------------------------------
Latch SR-NAND (S_n, R_n ativas em baixo):
  S_n=0 R_n=0 -> proibido (Q=QN=1)
  S_n=0 R_n=1 -> set   (Q=1)
  S_n=1 R_n=0 -> reset (Q=0)
  S_n=1 R_n=1 -> memória

Latch SR-NOR (S, R ativas em alto):
  S=0 R=0 -> memória
  S=0 R=1 -> reset
  S=1 R=0 -> set
  S=1 R=1 -> proibido (Q=QN=0)

Latch D (NAND ou NOR):
  EN=0, D=x  -> memória
  EN=1, D=0  -> Q=0
  EN=1, D=1  -> Q=1

FF D mestre-escravo (borda positiva/negativa):
  borda ativa + D=0 -> Q=0
  borda ativa + D=1 -> Q=1
  demais casos      -> Q mantém

FF JK (borda positiva):
  J=0 K=0 -> Q mantém
  J=0 K=1 -> Q=0
  J=1 K=0 -> Q=1
  J=1 K=1 -> Q = not Q (toggle)

FF T (borda positiva):
  T=0 -> Q mantém
  T=1 -> Q = not Q (toggle)

Em todos os FFs do Ex.4: CLRN=0 força Q=0; PRN=0 força Q=1 (assíncronos).
